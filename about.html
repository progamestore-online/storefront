About ProGameStore

Premium multiplayer web games with server-side state, cloud sync, and the same brand-consistent shell as FreeGameStore. Subscription-priced.

What this is

ProGameStore is a curated marketplace for browser games that legitimately need a backend — turn-based multiplayer, real-time lobbies, persistent worlds, server-side AI. Every game is a Progressive Web App; every game ships with the same UI shell as FreeGameStore; every game is subscription-priced with a transparent 10% platform commission.

Why a paid tier exists

Real-time multiplayer over WebSockets runs on Cloudflare Durable Objects. A popular multiplayer game with a thousand concurrent players burns ~$4,000/month in DO charges alone. The free tier on FreeGameStore doesn't try to absorb that cost; it ships single-player games only. ProGameStore is where the multiplayer story lives, and the subscription pays the bill.

Multiplayer is the headline

Every game here has a server-authoritative component. Turn-based vs. a remote opponent, real-time lobbies, persistent worlds — whatever the game needs.

Same shell as Free

The brand — fonts, layout, mute toggle, sign-in — is identical to FreeGameStore. A game's Pro version looks the same as its Free version; the difference is in what it does.

Cross-device sync

One @progamestore account; your games follow you. Save state in the cloud, not localStorage.

Quality-gated, not open source

Pro game code can be private. Same compliance audit as Free (fonts, viewport, no tracking) plus uptime + correctness gates on the multiplayer service.

Free vs. Pro

FreeGameStore

  • Free forever
  • Single-player only
  • Local-only state (no cloud)
  • MIT-licensed, open source
  • 43 games today
  • No accounts required to play

ProGameStore

  • Per-game subscription (e.g. $2/mo)
  • Real-time multiplayer + persistent state
  • Cloud save sync across devices
  • Server-side AI opponents
  • Curated catalog — quality-gated
  • 7-day free trial on every game

How publishing works

Game developers run npx -y @progamestore/cli@latest init my-game to scaffold from a template (turn-based, real-time, or 3D-persistent). The CLI ships a working Worker + Durable Object skeleton, the useRooms() client hook, and the same compliance gate as Free. pgs publish auto-provisions GitHub repo + Cloudflare Worker + DNS + Stripe Connect subscription. Developer sets the price; platform takes 10%.

Open?

The platform is open: progamestore-online/platform (SDK + CLI + compliance, MIT). Individual games on Pro can be private — that's the developer's call. The Free side (FreeGameStore) requires MIT for every listed game; Pro doesn't.

Status

ProGameStore launched in May 2026. The first listed game is multiplayer chess. The platform infrastructure (SDK, CLI, compliance suite, scaffolding templates) is feature-complete; the storefront catalog grows as more multiplayer games ship.
