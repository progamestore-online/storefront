Documentation

Everything you need to build a multiplayer game. One SDK, brand-consistent, scroll-free.

Quick Start

npm i @progamestore/games
import { GameShell, GameTopbar } from '@progamestore/games'

export default function App() {
  return (
    <GameShell topbar={<GameTopbar title="My Game" score={0} />}>
      {/* your game canvas */}
    </GameShell>
  )
}

GameShell

Root layout wrapper. Locks the viewport to 100svh, prevents document scroll, disables text selection. Every game wraps its content in this.

<GameShell topbar={<GameTopbar title="Chess" score={0} />}>
  <ChessBoard />
</GameShell>
PropTypeDescription
topbarReactNode?Topbar element (typically GameTopbar)
childrenReactNodeGame content, fills remaining viewport

GameTopbar

Standard topbar for all games. Same font, padding, and colors everywhere.

<GameTopbar title="Chess" stats={[
  { label: 'Score', value: 1200, accent: true },
  { label: 'Time', value: '3:42' }
]} />
PropTypeDescription
titlestringGame name (required)
scorenumber?Simple score display
statsGameTopbarStat[]?Custom stat chips

GameAuth

Platform-level sign-in button. Handles GitHub OAuth, displays avatar when signed in.

useRooms()

Real-time multiplayer via Cloudflare Durable Objects. The headline Pro feature.

import { useRooms } from '@progamestore/games'

function MultiplayerGame() {
  const { status, send, messages, peers } = useRooms({
    gameId: 'chess',
    roomId: 'lobby-1',
    onMessage: (msg) => handleMove(msg),
  })

  return (
    <div>
      <p>Status: {status}</p>
      <p>Players: {peers.length}</p>
      <button onClick={() => send({ type: 'move', data: { from: 'e2', to: 'e4' } })}>
        Move
      </button>
    </div>
  )
}
OptionTypeDescription
gameIdstringYour game's ID
roomIdstringRoom to join (or create)
onMessage(msg: RoomMessage) => voidCallback for incoming messages
ReturnTypeDescription
statusRoomStatus'connecting' | 'open' | 'closed' | 'error'
send(msg) => voidSend a message to all peers
messagesRoomMessage[]Message history
peersstring[]Connected peer IDs

useAuth()

Platform-level authentication hook.

const { user, signIn, signOut } = useAuth()
// user: { login: string, avatar_url: string } | null

useLeaderboard()

Persistent D1 leaderboard for your game.

const { entries, submit, loading } = useLeaderboard('chess')
// entries: LeaderboardEntry[] = [{ user, score, rank, ts }]
// submit(score: number): Promise<void>

useGameSounds()

Mute-aware sound effects. Respects the global mute toggle in GameTopbar.

const { play } = useGameSounds({
  move: '/sounds/move.mp3',
  capture: '/sounds/capture.mp3',
  win: '/sounds/win.mp3',
})

play('move') // no-op if muted

useSound()

Low-level sound hook for single effects.

const { play, muted } = useSound('/sounds/click.mp3')

Leaderboard

Drop-in leaderboard UI component.

<Leaderboard gameId="chess" limit={10} />
